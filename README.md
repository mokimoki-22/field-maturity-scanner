## Scientific Background

This tool is inspired by the feature design proposed in:

Ofer et al., 2024  
Heliyon  
https://doi.org/10.1016/j.heliyon.2023.e23781

It uses publication trends, review ratios, and patent signals as exploratory indicators of field maturity.

This tool is complementary to SciTrends: while SciTrends focuses on predicting future popularity, this tool emphasizes interpretable exploration of research field dynamics.

Note:  
This tool focuses on interpretability and does not implement the original predictive model.


## 科学的背景

本ツールは、以下の研究で提案されたツールに着想を得て開発されたものです。

Ofer et al., 2024  
Heliyon  
https://doi.org/10.1016/j.heliyon.2023.e23781

本ツールでは、論文動向、レビュー比率、および特許シグナルを用いて、研究分野の成熟度を探索的に評価します。

本ツールはSciTrendsと相補的な関係にあり、SciTrendsが将来の人気（popularity）の予測に焦点を当てているのに対し、本ツールは研究分野の動態を解釈可能な形で把握することに重点を置いています。

※本ツールは解釈可能性を重視した探索を目的としており、原論文で提案された予測モデル自体を実装したものではありません。
